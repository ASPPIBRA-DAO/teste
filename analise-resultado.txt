Este é um resumo da análise dos componentes 'Header' no projeto:

- **`HeaderSection`**: O componente de cabeçalho principal está localizado em `App/Frontend/src/layouts/core/header-section.tsx`. Ele é usado em todos os layouts.

- **`DashboardLayout`**: Usa `HeaderSection`. As páginas no diretório `App/Frontend/src/app/dashboard` usam este layout.

- **`MainLayout`**: Usa `HeaderSection`. As seguintes páginas usam este layout:
    - `/` (home)
    - `/about-us`
    - `/contact-us`
    - `/faqs`

- **`SimpleLayout`**: Usa `HeaderSection`. As seguintes páginas usam este layout:
    - `/coming-soon`
    - `/maintenance`
    - `/payment`
    - `/pricing`

- **`AuthSplitLayout`**: Usa `HeaderSection`. As seguintes páginas usam este layout:
    - `/auth/jwt/sign-in`
    - `/auth/jwt/sign-up`
    - `/auth/supabase/sign-in`
    - `/auth/supabase/sign-up`
    - `/auth/supabase/reset-password`
    - `/auth/supabase/update-password`
    - `/auth/supabase/verify`
    - `/auth-demo/centered/sign-in`
    - `/auth-demo/centered/sign-up`
    - `/auth-demo/centered/reset-password`
    - `/auth-demo/centered/update-password`
    - `/auth-demo/centered/verify`
    - `/auth-demo/split/sign-in`
